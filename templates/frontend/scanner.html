NFC-Tag lesen

Aktiver NFC-Scan wird von diesem Browser nicht unterstützt.

iPhone (iOS 13+): NFC-Tags werden automatisch vom System erkannt — einfach das iPhone an den Tag halten. iOS zeigt ein Banner an, das den Link direkt in Safari öffnet. Dafür muss der Tag eine URL enthalten (z.B. https://…/geraete/42/).

Android: Aktiver Scan ist in Google Chrome ab Version 89 verfügbar. Bitte nutzen Sie Chrome auf Android oder den QR-Code-Scanner.

NFC – Gerätekompatibilität

  • iPhone (iOS 13+): Kein Scan-Button nötig — iPhone erkennt Tags automatisch und zeigt ein System-Banner an.
  • Android (Chrome 89+): Scan über den „NFC-Scan starten"-Button, HTTPS erforderlich.
  • Der NFC-Tag muss eine URL im Format /geraete/<id>/ oder eine reine Geräte-ID enthalten.
  • Alle anderen Browser: QR-Code-Scanner oder manuelle Suche verwenden.
